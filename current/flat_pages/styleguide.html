Flexible grid system

The class stack is:

<div class="mod-gridunit gu-[position] gu-last gu-[width]">
	<p>Content</p>
</div>
Class name What it does Value(s)
.mod-gridunit
  • Adds a right-hand margin of 2%
  • Ensures the element is set to display:block
  • Adds box-sizing:border-box to ensure borders etc… won't be included in it's width.
 .mod-gridunit
